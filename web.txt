<!doctype html> - html 문서라는 설명

<strong> - 굵게
<u> - 밑줄
<h1~h6> - 제목
<p> - 문단구분
<br> - 줄바꿈
<img> - 사진 불러오기
<li> - 목록
<ul> - 목록 나누기
<ol> - 목록 나누기(순서 포함)
<title> - 웹 제목 설정
<head> - 웹 페이지 설정 코드 범위
<body> - 본문 범위
<html> - html코드 범위
<a> - 링크 불러오기

src="" - 사진 주소
width="" - 사진 크기
href="" - 링크 주소
